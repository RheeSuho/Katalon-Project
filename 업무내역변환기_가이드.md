# 하루 업무 내역 정리 — 사용 가이드

업무 내역 페이지에서 복사한 텍스트를 붙여넣으면, 스프린트별로 자동 정리된 표를 구글 시트에 바로 붙여넣을 수 있습니다.

---

## 시작하기

`하루 업무 내역 정리.html` 파일을 더블클릭해 브라우저로 열면 됩니다.  
별도 설치 불필요. 인터넷 연결만 있으면 동작합니다.

---

## 사용 방법

### 방법 1 — 텍스트 붙여넣기 (권장)

1. 업무 내역 조회 페이지에서 내용 전체 복사 (`Cmd+A` → `Cmd+C`)
2. **텍스트 붙여넣기** 탭의 입력창에 붙여넣기 (`Cmd+V`)
3. **변환하기** 버튼 클릭
4. 미리보기에서 결과 확인

> 카카오웹툰, 타파스, 카카오페이지 등 여러 서비스를 한 번에 붙여넣어도 됩니다.  
> 서비스별 필터 버튼으로 전환할 수 있습니다.

### 방법 2 — CSV 파일 업로드

1. **CSV 파일 업로드** 탭 선택
2. 업무 내역 CSV 파일을 드래그하거나 클릭해서 선택
3. 자동으로 변환됩니다

---

## 자동 처리 규칙

| 항목 | 처리 방식 |
|------|-----------|
| `26-1차`, `KP-2026-4차` 등 스프린트 번호 패턴 | **정기 스프린트**로 분류, 차수 순 정렬 |
| 스프린트 번호 없는 항목 | **운영 지원**으로 분류 |
| 주간 회의 항목 | 결과에서 **자동 제외** |
| `ㄴ` 또는 `-` 로 시작하는 서브노트 | 바로 위 항목 뒤에 **이어붙임** |
| 진행 업무 띄어쓰기 차이 (`SN리뷰` / `SN 리뷰`) | **동일 항목으로 통합** |
| URL이 여러 개인 항목 | 각 URL 행에 수행 업무 **자동 반복** |

### 출력 컬럼 구조

```
구분 | 프로젝트 | 진행 업무 | 수행 업무 | 참고 문서
```

- **구분**: 정기 스프린트 / 운영 지원
- **프로젝트**: 스프린트 차수 (예: 26-1차) — 운영 지원은 빈 칸
- **진행 업무**: TC수행, adhoc, CL수행 등 업무 유형
- **수행 업무**: 실제 수행한 업무 내용
- **참고 문서**: 링크 (여러 개일 경우 각 행에 분리)

### 상단 요약 통계

변환 후 상단에 아래 항목이 표시됩니다.

| 항목 | 설명 |
|------|------|
| 서비스 수 | 감지된 서비스 수 (2개 이상일 때만 표시) |
| 전체 업무 | 수행 업무 총 항목 수 |
| 정기 스프린트 | 서비스별 고유 스프린트 차수 합계 |
| 스프린트 업무 | 정기 스프린트 소속 수행 업무 항목 수 |
| 운영 지원 | 운영 지원 소속 수행 업무 항목 수 |

## 결과 활용

### 구글 시트에 붙여넣기

1. **📋 복사** 버튼 클릭
2. 구글 시트에서 붙여넣기 (`Cmd+V`)
3. 구분 · 프로젝트 · 진행 업무 열이 **병합 셀**로 자동 적용됨

### 파일 다운로드

| 버튼 | 설명 |
|------|------|
| 📄 CSV 다운로드 | 한글 깨짐 없는 UTF-8 CSV |
| 📊 Excel 다운로드 | 셀 병합 적용된 .xlsx 파일 |

> **문서 제목 설정**: 다운로드 전 "문서 제목" 입력란에 원하는 제목을 입력하면 파일명과 시트 상단 제목에 반영됩니다.
